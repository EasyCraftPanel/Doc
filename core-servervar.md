# 服务器变量

在服务器核心配置中, 你可以使用服务器变量. 我们所提供的服务器变量有

* `SERVERID`: 服务器ID

* `SERVERDIR`: 服务器路径

* `CORE`: 核心ID

* `PORT`: 服务器端口

* `PLAYER`: 最大玩家数

* `WORLD`: 默认世界

* `REMOVE`: 移除此参数置空

* `ENV`: 环境文件夹

  指向 `EasyCraft目录/environment/`,这里存放运行环境, 比如 PHP , Java

  为了统一我们推荐使用以下的子路径

  * php5
  * php7
  * php72
  * php8
  * jdk8
  * jdk7
